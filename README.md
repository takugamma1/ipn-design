# IPN България — Web Mockup

Интерактивен прототип на новия сайт на **IPN България** (консумативи за печат).
Статичен сайт (HTML + Tailwind CSS), напълно адаптивен (desktop + mobile),
изграден за презентация пред клиента.

## 🔗 Live demo
**https://takugamma1.github.io/ipn-design/**

## Страници
| Страница | Файл | Описание |
|----------|------|----------|
| Начало | [index.html](index.html) | Начална страница на магазина |
| Каталог | [collection.html](collection.html) | Листинг (XEROX оригинални консумативи) с филтри |
| Продукт | [product.html](product.html) | Продуктова страница — адаптивна (desktop и mobile) |
| Бизнес портал | [dashboard.html](dashboard.html) | B2B клиентски панел |

## Навигация в демото
- **Начало → Каталог**: бутон „Покажи консумативите", картите на марките, „Виж всички"
- **Каталог → Продукт**: бутон „Виж детайли"
- **Начало → Бизнес портал**: бутон „Вход"

## Технологии
- HTML5 + [Tailwind CSS](https://tailwindcss.com) (CDN)
- Inter (Google Fonts), кирилица
- Логото е векторно (SVG) — остро на всякакъв размер
- Реални продуктови снимки в `images/`
